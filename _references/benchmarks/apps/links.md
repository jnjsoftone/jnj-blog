```markdown

- **카테고리**: [Productivity / Social / E-commerce / etc.]
- **분석 날짜**: YYYY-MM-DD
- **다운로드**: [앱스토어 링크]
- **분석 포인트**:
  - 핵심 기능 1
  - UX 패턴 2
- **스크린샷**: ./screenshots/app-name-feature.png
- **기능 분석**: ./features/app-name-features.md
- **적용 아이디어**:
  - [ ] 적용할 기능 1
  - [ ] 적용할 UX 패턴 2
```
