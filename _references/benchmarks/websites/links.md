# Benchmark Websites

벤치마크할 웹사이트 목록과 분석 내용을 기록합니다.

---

## 템플릿

각 사이트를 추가할 때 아래 템플릿을 사용하세요:

```markdown
### [사이트 이름]
- **URL**: https://example.com
- **카테고리**: [E-commerce / SaaS / Portfolio / etc.]
- **분석 날짜**: YYYY-MM-DD
- **분석 포인트**:
  - 핵심 기능/디자인 요소 1
  - 핵심 기능/디자인 요소 2
- **스크린샷**: ./screenshots/site-name-feature.png
- **상세 분석**: ./analysis/site-name-analysis.md
- **적용 아이디어**:
  - [ ] 적용할 기능/디자인 1
  - [ ] 적용할 기능/디자인 2
```
