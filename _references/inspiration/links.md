## 🎨 UI/UX Pattern Collections

### Mobbin
- **URL**: https://mobbin.com/
- **설명**: 모바일 앱 UI 디자인 갤러리 (10만+ 스크린샷)
- **카테고리**: 로그인, 온보딩, 프로필, 검색 등
- **활용**: 모바일 UI 패턴 참고

### UI Patterns
- **URL**: http://ui-patterns.com/
- **설명**: 검증된 UI 패턴 라이브러리
- **특징**: 패턴별 설명, 사용 사례, 예시
- **활용**: 일반적인 UI 문제 해결

### Dribbble
- **URL**: https://dribbble.com/
- **설명**: 디자이너 포트폴리오 및 디자인 쇼케이스
- **활용**: 최신 디자인 트렌드 파악

### Behance
- **URL**: https://www.behance.net/
- **설명**: Adobe의 크리에이티브 작품 공유 플랫폼
- **활용**: 완성도 높은 프로젝트 사례
